# ВЕРТУМ CRM - Торговая система

## 📊 О системе
Полнофункциональная CRM система для управления клиентами, визитами и отчетами.

## 🚀 Функционал
- 📋 Управление клиентами с фильтрами и поиском
- 📅 Календарь встреч и визитов
- 📊 Генерация и отправка отчетов
- 📧 Интеграция с почтой (Yandex, SMTP)
- 📁 Импорт/экспорт Excel файлов
- 👥 Управление менеджерами

## 🛠️ Установка
1. Клонировать репозиторий
2. Установить зависимости: `npm install`
3. Создать файл `.env` на основе `.env.example`
4. Запустить: `node server.js`

## ⚙️ Настройка почты
Создайте файл `.env`:
EMAIL_USER=ваш_email@example.com
EMAIL_PASSWORD=ваш_пароль
EMAIL_HOST=smtp.yandex.ru
EMAIL_PORT=465
EMAIL_SECURE=true
